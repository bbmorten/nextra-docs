## Difference in days

```python
from datetime import date

# Define two dates
date1 = date(2024, 12, 15)  # Year, Month, Day
date2 = date(2024, 12, 10)

# Calculate the difference
difference = date1 - date2

# Output the difference in days
print(difference.days)  # Output: 5
```
