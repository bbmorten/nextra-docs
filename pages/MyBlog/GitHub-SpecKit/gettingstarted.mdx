# Getting Started with GitHub SpecKit

This guide will help you get started with GitHub SpecKit, a powerful tool for managing and documenting your GitHub repositories. Follow the steps below to set up and start using GitHub SpecKit.

## Links

- [GitHub SpecKit Repository](https://developer.microsoft.com/blog/spec-driven-development-spec-kit?)


## Installation Steps


```shell
uvx --from git+https://github.com/github/spec-kit.git specify init {ProjectName}

uvx --from git+https://github.com/github/spec-kit.git specify init spec-kit-v1
```

- Follow the instructions to start your project
